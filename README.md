## Features
#### Backend

- Proyecto modularizado
- Desplegado en Cloud Functions Firebase
- Uso de express-validator
- Middleware

#### Frontend

- Proyecto modularizado
- Permite ser escalable
- Libreria PrimeNg para estilos
- Lazy Loading
- Permite enrutar mas modulos y presentarlos
